If you want to read more about this website and how I created it, navigate to the Tech Projects page to do so, and check out my other projects while you're there! I have also created a Music Projects page to talk about my musical background as well as the arrangements that I have written.

My Personal AI Policy

The whole point of me taking on all of these personal projects is for me to learn new skills myself. With that in mind, I have not and will not ever use code generated by an LLM for any of my projects. I will reference online sources to learn how to write my own code and nothing else. However, I understand that knowing how to properly use AI is an important skill itself, so I will (when I am particularly stuck on something) ask AI for tips, but NEVER copy/paste ANY generated code. I take pride in accomplishing things on my own, and to me the rewarding feeling of completing a project is partially lost if half the code was AI generated.
