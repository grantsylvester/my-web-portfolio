This Website

The first personal project I would like to cover is this website that you yourself are currently browsing! I decided to make this website for several reasons. First, I wanted to learn HTML, CSS and Javascript in case I ever end up needing to design or at least understand a website or web application in my future.

Since I was interested in learning how to make a website, I decided that it would be a cool way to make a project portfolio where I can yap about all of my interests. And here we are!

I made this website with GitHub Pages (its free). To learn how to host a website from GitHub, I followed GitHub's tutorial and made a test website. This is not that website, as the tutorial uses markdown formatting instead of HTML. To learn HTML, I followed W3 School's guide. They have a well structured online reference as well as short quizzes at the end of every chapter, which I found super helpful (and its also free).

I intend to keep working on this site to make it look prettier and also involve some Javascript somewhere. One thing that I would like to add is a collapsible navigation bar, or perhaps some other smooth transitions.
